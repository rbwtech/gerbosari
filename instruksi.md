_PENGUMUMAN TUGAS AKHIR KELOMPOK (TA)_

Judul Proyek: "Rancang Bangun Website Profil Kelurahan Berbasis Cloud di Wilayah Yogyakarta"

_*Untuk kelompoknya sama dengan kelompok sebelumnya*_

1. Deskripsi Tugas :
   Setiap kelompok diwajibkan membangun sebuah website profil desa/kelurahan yang berlokasi di sekitar Daerah Istimewa Yogyakarta (DIY). Website ini wajib di-hosting menggunakan layanan cloud (misal: AWS, Google Cloud, Azure, Vercel, Netlify, atau DigitalOcean).

2. Komponen Website (Statis & Dinamis)
   Website harus terdiri dari dua bagian utama:
3. Konten Statis (Profil Utama): Mengambil data dari Wikipedia atau situs resmi kelurahan terkait. Konten meliputi: Sejarah Desa, Visi Misi, Struktur Organisasi, dan Peta Wilayah.

4. Konten Dinamis (Fitur Tambahan): Dibuat menggunakan database (MySQL/PostgreSQL/MongoDB) dan bahasa backend (PHP/Node.js/Python). Fitur wajib:
   Update Lowongan Kerja (UMKM/Lokal).
   Berita/Agenda Kegiatan Kelurahan Terbaru.
   Galeri Foto/Video Kegiatan Desa.

Pembagian tugas, desa yang kita pilih:
Buat instance baru AWS pakai akun addin
isi web (info dari wikipedia, buku, gerbosari-kulonprogo.desa.id dan berbagai website lain)

Isinya:

1. Statis (svelte, vite, bisa anda sesuaikan lagi best practice gimana): Sejarah desa, visi misi, struktur organisasi, peta wilayah
2. Dinamis (rust backend dan database mysql): Galeri foto kegiatan desa, data penduduk

data-data di atas sudah ada di _.docx dan _.pdf yang di folder sama ini, anda bisa fetch image yang dibutuhkan dalam docx lalu pisahkan menjadi file terpisah untuk galeri foto kegiatan desa dan lain-lain.
